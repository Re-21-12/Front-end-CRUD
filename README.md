## Front-end-CRUD

This development was developed with the following tools:
- JavaScript (TypeScript)
- Node (Pnpm)
- CSS (Tailwind CSS)
- React (ViteJS)
- API client (Axios)
- React Router DOM

Currently, this development doesn't have a deployment on any service.
